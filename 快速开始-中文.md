# Linux 无 Logo 版极简操作说明

这个版本与标准 Linux 包功能一致，唯一差异是不包含 logo.png，页头会自动显示文字标识。

## Docker 方式

1. 进入当前目录
2. 执行：bash start-docker.sh
3. 浏览器访问部署主机的访问地址和端口

## 物理机方式

1. 准备可访问的 MySQL
2. 首次运行前会自动生成 .env.local
3. 按实际环境补齐数据库连接
4. 执行：bash start-local.sh

## 常用操作

- 状态：bash status-local.sh / bash status-docker.sh
- 日志：bash logs-local.sh / bash logs-docker.sh
- 停止：bash stop-local.sh / bash stop-docker.sh

## 服务安装

- 本机服务：bash install-service-local.sh
- Docker 服务：bash install-service-docker.sh
