# BioControl 极简操作说明

## 1. 下载

下载与你系统对应的平台包并解压。

## 2. 选一种启动方式

### 方式 A：Docker

- Linux/macOS：执行 `bash start-docker.sh`
- Windows：执行 `powershell -ExecutionPolicy Bypass -File .\start-docker.ps1`

首次执行会自动生成 `.env`。

### 方式 B：物理机

先准备可访问的 MySQL，然后：

- Linux/macOS：执行 `bash start-local.sh`
- Windows：执行 `powershell -ExecutionPolicy Bypass -File .\start-local.ps1`

首次执行会自动生成 `.env.local`，按实际环境补齐数据库连接即可。

## 授权说明

- 安装后自动进入 1 个月试用期
- 试用期间可正常使用，不需要先激活
- 如需正式授权，在系统内打开激活界面，复制显示的物理地址（MAC）并发送给管理员

## 3. 查看状态与日志

- 状态：`status-local` 或 `status-docker`
- 日志：`logs-local` 或 `logs-docker`

## 4. 停止

- 本机：`stop-local`
- Docker：`stop-docker`

## 5. 开机自启

仅 Linux 支持：

- 本机服务：`install-service-local.sh`
- Docker 服务：`install-service-docker.sh`
