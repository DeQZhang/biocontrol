# Windows 极简操作说明

## Docker 方式

1. 进入当前目录
2. 执行：powershell -ExecutionPolicy Bypass -File .\start-docker.ps1
3. 浏览器访问部署主机的访问地址和端口

## 物理机方式

1. 准备可访问的 MySQL
2. 首次运行前会自动生成 .env.local
3. 按实际环境补齐数据库连接
4. 执行：powershell -ExecutionPolicy Bypass -File .\start-local.ps1

## 常用操作

- 状态：powershell -ExecutionPolicy Bypass -File .\status-local.ps1 / powershell -ExecutionPolicy Bypass -File .\status-docker.ps1
- 日志：powershell -ExecutionPolicy Bypass -File .\logs-local.ps1 / powershell -ExecutionPolicy Bypass -File .\logs-docker.ps1
- 停止：powershell -ExecutionPolicy Bypass -File .\stop-local.ps1 / powershell -ExecutionPolicy Bypass -File .\stop-docker.ps1

## 服务安装

- 本机服务：powershell -ExecutionPolicy Bypass -File .\install-service-local.ps1
- Docker 服务：powershell -ExecutionPolicy Bypass -File .\install-service-docker.ps1
