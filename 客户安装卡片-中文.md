# BioControl 客户安装卡片

## 第一步：下载

按系统下载对应文件：

- Linux：`biocontrol-linux-20260420.tar.gz`
- Linux 无 Logo：`biocontrol-linux-no-logo-20260420.tar.gz`
- macOS：`biocontrol-mac-20260420.tar.gz`
- Windows：`biocontrol-win-20260420.zip`

## 第二步：启动

### Docker

- Linux/macOS：`bash start-docker.sh`
- Windows：`powershell -ExecutionPolicy Bypass -File .\start-docker.ps1`

### 物理机

- Linux/macOS：`bash start-local.sh`
- Windows：`powershell -ExecutionPolicy Bypass -File .\start-local.ps1`

启动后默认先进入 1 个月试用期；如需正式授权，请在系统激活界面查看物理地址（MAC）并发送给管理员。

## 第三步：停止

- Linux/macOS 本机：`bash stop-local.sh`
- Linux/macOS Docker：`bash stop-docker.sh`
- Windows 本机：`powershell -ExecutionPolicy Bypass -File .\stop-local.ps1`
- Windows Docker：`powershell -ExecutionPolicy Bypass -File .\stop-docker.ps1`
